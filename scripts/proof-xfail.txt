# proof-xfail.txt — allowlist of acceptable unproved SPARK checks.
#
# Each non-comment line is a substring pattern. An unproved check is ACCEPTED
# if its identity (as printed by `scripts/check-proof.sh --list`) contains any
# pattern below; any unproved check matching none of them fails the gate as a
# regression. See scripts/check-proof.sh for the full rationale.
#
# INTENTIONALLY EMPTY. Under the GNAT/GNATprove/SPARKlib 16 toolchain the whole
# memcp crate proves to Silver with no escape hatches — including the SPARKlib
# floating-point range lemmas that the 15-series prover set could not discharge.
# The gate therefore fails on ANY unproved check. If a genuine upstream gap
# resurfaces on some platform, add a substring pattern here (see it with
# `scripts/check-proof.sh --list`); until then the allowlist stays empty so the
# gate demands a fully clean proof.
