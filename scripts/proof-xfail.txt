# proof-xfail.txt — allowlist of acceptable unproved SPARK checks.
#
# Each non-comment line is a substring pattern. An unproved check is ACCEPTED
# if its identity (as printed by `scripts/check-proof.sh --list`) contains any
# pattern below; any unproved check matching none of them fails the gate as a
# regression. See scripts/check-proof.sh for the full rationale.
#
# The only accepted gaps live in SPARKlib's floating-point range lemmas
# (SPARK.Lemmas.Float_Arithmetic / Long_Float_Arithmetic .Is_Float). They need
# the COLIBRI solver, which the default prover set does not ship, so they are
# never discharged — and which of them show up is platform-dependent (the arm64
# dev machine clears more than the x86_64 CI runner). Matching on the shared
# lemma source file covers every such variant. None of these are memcp code.
